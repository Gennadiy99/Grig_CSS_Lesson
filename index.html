Свойства для grid-container

Диву присвается класс "grid-container"

В css для "grid-container" указывается стиль "display:grid"

grid-template-columns: 1fr 1fr 1fr; - для создания количества и размера колонок.

Значения для grid-template-columns:
  • 150px 150px 150px = 3 колонки по 150px Шириной.
  • 20% 20% 20% = 3 колонки шириной по 20% каждая
  • 1fr 1fr 1fr = 3 колонки шириной по 1fr каждая (1fr - это 1часть свободного пространства)
  • repeat(3, 1fr) = Функция которфя повторяет 3 раза колонку размером в 1fr
  • repeat(3, minmax(150px, 1fr))= Функция "repeat" которая повторяет 3 раза,функцию "minmax"которая устанавливает мин. и макс. ширину для колонки.
    • grid-template-columns: repeat(auto-fit, minmax(100px,1fr)); "auto-fit" - размещает колонки пока хватает места в контейнере основываясь на минимальный размер "minmax" если места не хватает то колонки переносятся.При избытке места занимает все пространство Контейнера увеличивая ширину колонки на значение "max - 1fr"

Свойства для "grid-item" - "элементов сетки" для КОЛОНОК

Элемент "grid" контейнера с классом .item в CSS описываются свойства:

  • grid-column-start: 1; - Первая границы колонки
  • grid-column-end: 4; - Вторая граница колонки
  • grid-column-end: span 3; - "span"- Обозначает кол-во "пространств"
  • grid-column: 2/4; - Можно описать "Первую гр.кол" / "Вторую гр.кол."
  • grid-column: 2/span 2; - Можно описать "Первую гр.кол." / "Кол-во пространств"

Свойство устанавливает промежуток между границами элемента "зазор"

  • grid-row-gap: 5px; - Зазор в 5px между строк
  • grid-column-gap: 5px; - Зазор в 5px между колонками
  • grid-gap: 10px; - Зазор между Кол. и Строками 10px
  • grid-gap: 10px 5px; - Зазор между строками 10px между колонками 5px

При создании в контейнере строки или колонки без размера, размер бедется из свойств:

  • grid-auto-rows: 100px;
  • grid-auto-columns: 1fr;

Устанавливает направление элементов в сетке:

  • grid-auto-flow: column; Элементы будут идти по очередно с верху в низ (в колонке.)
  • grid-auto-flow:row; Элементы будут идти с лево на право (строкой)

Рассмотрим свойство "grid-template-areas" / "grid-area"

  • "grid-template-areas" - Указывается для контейнера (Шаблон области сетки)
    • "head head head head" - 1-я Область строка из 4х колонок привязанных к элементу "head"
    • "main main main sidebar" - 2-я Область строка из 4х кол. прив. к 2м элементам "main" / "sidebar"
    • "footer footer footer sidebar" - 3-я Область строка из 4х кол. прив. к 2м элементам "footer" / "sidebar"
  • "grid-area" - Указывается для элемента ".box-text1": 
              .box-text1 {
            grid-area: head;
          }

Рассмотрим выравнивание: justify-items / justify-content / justify-self

justify-items - Свойства для контейнера который позволяет выравнивать элементы ВНУТРИ ЯЧЕЕК.

    • justify-items - Выравнивание элементов по горизонтали.
    • align-items - Выравнивает по вертикали
    • place-items - Обьеденяет в себе 2 выравнивания Вертикаль и горизонталь
  2. justify-content - выравнивание Всех элементов внутри контейнера
  3. justify-content: space-around; - Выравнивание всего Контента с равными отступами.
  4. justify-content: space-between; - Выравнивание всего Контента с равными отступами.Но вначале и конце нет отступов.
  5. justify-content: space-evenly; - Выравнивание всего Контента с равными отступами.
  6. align-content - Выравнивание Всех элементов по вертикали
  7. Все свойства "justify-content" Горизонтальное выравнивание - ВСЕХ ЭЛЕМЕНТОВ в КОНТЕЙНЕРЕ.
    • justify-content:start;
    • justify-content:end;
    • justify-content:center;
    • justify-content:space-around;
    • justify-content:space-between;
    • justify-content:space-evenly;
  8. Все свойства "align-content" Вертикальное выравнивание - ВСЕХ ЭЛЕМЕНТОВ в КОНТЕЙНЕРЕ.
    • align-content:start;
    • align-content:end;
    • align-content:center;
    • align-content:space-around;
    • align-content:space-between;
    • align-content:space-evenly;
  9. Совместное свойство "place-content" для Вертикального и Горизонтального выравнивания - ВСЕХ ЭЛЕМЕНТОВ в КОНТЕЙНЕРЕ.
  10. Выравнивание для самих элементов: "justify-self" / "align-self" / "place-self"
    1. "justify-self:"
      • justify-self: start;
      • justify-self: center;
      • justify-self: end;
      • place-self:
    2. "align-self"
    3. "place-self"
